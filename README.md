Digital Locker

A secure web application that allows users to upload, store, manage, and access their important digital documents in one place.

# Project Overview

Digital Locker is a document management system where users can upload files, categorize them, set expiry dates, preview documents, and manage them easily through a simple dashboard interface.

The project demonstrates full-stack development using Node.js, Express.js, MongoDB, and EJS.

---

# Features

- 📤 Upload documents
- 📂 Categorize files (Personal, Identity, Education, Financial, Medical, etc.)
- 🗓 Add expiry dates to documents
- 🔍 Search files instantly
- 👀 Preview documents
- ✏ Edit file category and expiry date
- 📝 Rename uploaded files
- ⬇ Download files
- 🗑 Delete files
- 🔔 Toast notifications for actions

# Installation

Clone the repository:

git clone https://github.com/YOUR_USERNAME/digital-locker.git

Go to the project directory:

cd digital-locker

Install dependencies:

npm install

Run the server:

node app.js

Open in browser:

http://localhost:3000
